CN_GL - FPS Example (CS456 Final Project)

Shaders

  1. Real Time Dynamic Water Reflections - The scene renders the entire world in reverse, cuts off all drawing (in the reflected world) under the water height, and then assigns that to a framebuffer. Then, in the fragment shader, the tex coordinates are convered into screen coordinates and drawn onto the screen only where the water is. There are no ripples (yet). This is a perfect reflection. It should be noted that you can see through the water since it is drawn with transparency. This demonstrates WebGL's blending modes.
  2. Real Time Dynamic Shadows - The scene renders the entire world in perspective projection from the light source, creates a depth map, and then uses that to tell whether or not a point in 3D space is supposed to be shaded. If it is, all lighting is disabled on that pixel and instead, it takes that pixel and multiples its RGB values by 0.5 and draws that. It lacks anti-aliasing, so it does look jagged on the edges.
  3. Phong Shading - Somewhat inaccurate, but gets the job done. Light dynamically interacts with the environment to light up parts that it should be reflecting off of.
  4. Specular Lighting - Subtracts from the texture and then multiplies back to simulate a "Levels" effect to get only the brightest colours out. Then, Phong shading only affects those pixels as opposed to the darker coloured ones.
  5. 2 Pass Fast Blur - Two pass fake Gaussian Blur via shaders. The first pass comes from rendering the entire scene. The fragment shader will blur it on the X axis only. Afterwards, that one is drawn to the next framebuffer, where the second fragment shader will blur it on the Y Axis. It is then drawn to the screen using GL Blending. This is the last step in the drawing process.

Objects/Effects

  1. Mario Cube x3 - The only logical excuse to have a cube flying in the air. Made it spin to show basic animations off.
    • Includes Phong Shading w/ Texture
    • Affected by Shadows
  2. Teapot - It's cute, so I added it and I hid it in the map. Try to find it without cheating. :)
    • Includes Phong Shading w/ No Texture
    • Affected by Shadows
  3. Railgun - Straight from Quake. Again, try finding this one. And then try picking it up. :) (Spoilers in the next section)
    • Affected by Shadows
  4. UX2 Map - This shouldn't really be counting as an object, but the world is split up into the 3 materials that make it.
    • Sand - Phong Shading
    • Metal - Phong Shading
    • Brick - (Awkwardly) Phong Shading
    • All of which are affected by Shadows
  5. Water - Not trying to double dip here, but the water is actually loaded from an OBJ file that has a 1x1x0 plane. This plane is then set to cast the reflected world on.
    • Affected by Shadows
    • Casts dynamic reflection of the world

Other Features

  1. Completely rewritten engine from scratch - Based off of my DERPG UX2 Engine I wrote back in 2013, CN_GL is a WebGL engine made in under 96 hours to conquer a homework assignment and also allow me to write WebGL applications in the near future much more easily than without it.
  2. FPS-like control scheme - WASD to move, captures the mouse (Chrome and Firefox only) to allow you to look around with it without actually moving the cursor. Shoot Railgun with left mouse click.
  3. Dynamically Resizable Window - Resize the window! Go on, I dare you. It won't change how the game performs.
